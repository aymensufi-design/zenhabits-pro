# 🧘 ZenHabits Pro v2.0
> **A Smart Personal Productivity & Mental Well-being Tracker**

ZenHabits Pro ek comprehensive tool hai jo users ko unke daily tasks manage karne ke sath-sath emotional health track karne mein madad karta hai. Ise **Python** aur **Streamlit** ka use karke banaya gaya hai.

---

## 🚀 Key Features

* **📊 Smart Mood Analytics:** NLP ka use karke aapke daily journal entries ko analyze karta hai aur mood trends visualize karta hai.
* **🎉 Instant Rewards:** Task complete hone par real-time animations (balloons) aur celebratory messages deta hai.
* **📈 Progress Tracking:** Daily progress charts aur "Mastery Score" ke sath productivity check karein.
* **🗑️ Task Management:** Tasks ko add, search, aur delete karne ka seamless experience.
* **🔒 SQLite Database:** Aapka sara data aur journals locally safe aur organized rehte hain.

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit (Python Framework)
- **Visualization:** Plotly & Plotly Express
- **Database:** SQLite3
- **Data Handling:** Pandas & Python Logic
